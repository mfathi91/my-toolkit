# My Toolkit

A collection of web-based utility tools.

## Available Tools

### Video Compressor
- Hardware-accelerated HEVC (H.265) compression
- Two compression modes: Standard and Deep
- Platform detection (Apple Silicon / Intel Quick Sync / Software fallback)
- Automatic file cleanup after download

## How to Run

1. Copy `docker-compose.yml` to your server
2. Run the application:

```bash
docker compose up -d
```

Access at: http://localhost:8765
